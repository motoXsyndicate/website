MOTOXSYNDICATE MXS LIVE - QUICK SETUP
======================================

The live page is located at:
https://www.motoxsyndicate.com/live/

EDIT BEFORE EACH BROADCAST
--------------------------
Open this file with NOTEPAD (do not double-click it):
static/live-config.js

1. Schedule your YouTube Live stream.
2. Copy the video ID from the YouTube URL.
   Example URL: https://www.youtube.com/watch?v=ABC123xyz
   Video ID: ABC123xyz
3. Paste it into:
   youtubeVideoId: "ABC123xyz",
4. Change status to:
   status: "starting", before the stream
   status: "live", when you go live
   status: "intermission", between motos
   status: "ended", when finished
5. Update the race title, track, class, date, start time, schedule,
   registration URL, results URL, ticker, next event, and sponsors.
6. Save the file.
7. In Command Prompt run these commands ONE AT A TIME:

   cd C:\MXS-Website
   git add .
   git commit -m "Update MXS Live broadcast"
   git push

Cloudflare will automatically deploy the update.

YOUTUBE CHAT
------------
YouTube chat appears automatically as a tab when a video ID is entered.
Live chat must be enabled for the scheduled YouTube broadcast.

COUNTDOWN TIME
--------------
Use a complete date/time with the timezone offset.
Houston Central Daylight Time example:
startTime: "2026-07-26T19:00:00-05:00",

Houston Central Standard Time example:
startTime: "2026-12-06T19:00:00-06:00",
