MXS LIVE GOOGLE SHEET CONTROL PANEL
===================================

The Live page checks the published Google Sheet every 10 seconds.
After this website version is deployed, normal race-night changes do NOT
require VS Code, GitHub edits, or another Cloudflare deployment.

Keep labels in column A and enter values in column B.
Labels are not case-sensitive.

CORE ROWS
---------
A1  Event Name             B1  MXS Outdoor Round 2
A2  Race Date              B2  July 19, 2026
A3  Race Time              B3  7:00 PM
A4  Registration URL       B4  https://...
A5  YouTube URL            B5  https://youtube.com/watch?v=...

IMPORTANT
---------
1. The Google Sheet must remain published to the web as CSV.
2. Press Enter after changing a cell.
3. Existing viewers receive the update within about 10 seconds.
4. Blank optional cells fall back to the defaults in live-config.js.
5. Use a full YouTube watch/live URL with a video ID to embed the stream.
   A channel URL can open in a button but cannot be embedded automatically.
