MXS LIVE GOOGLE SHEET SETUP

1. Open your Google Sheet.
2. Rename the first sheet tab to: Live
3. Paste the two-column table below into cell A1.
4. In Google Sheets, go to File > Share > Publish to web.
5. Publish the "Live" sheet as a web page, then confirm.
6. Replace these two website files:
   C:\MXS-Website\static\live-config.js
   C:\MXS-Website\static\live.js
7. Push the changes to GitHub once.

After that, edit the Google Sheet for every race. No more code changes or Git pushes.

YOUTUBE:
Paste the full YouTube livestream URL into youtubeUrl.
When youtubeUrl is blank, the button opens the MXS YouTube channel instead.

REGISTRATION:
Paste the full Google Form or registration page URL into registrationUrl.
Leave it blank to hide the Race Registration button.
